## Your lab environment

Each team has a set of dedicated resources as well as access to global shared systems:

- Shared Azure subscription accessed through your team user account
- Your dedicated vWAN hub connected to shared global Virtual WAN
- A **branch site** simulating an on-premises office — FortiGate + Windows desktop
- A regional **spoke VNet** with a simple web server
- shared FortiManager with dedicated ADOM per each team

You will access all resources using the same username and password with small exception of Azure which requires full principal name ("vwanlabXX@fortinetcloud.onmicrosoft.com").

Your goal over the course of this workshop is to connect all these pieces together into a working SD-WAN deployment.
