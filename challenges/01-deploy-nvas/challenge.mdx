## Deploy NVAs

Use Azure Marketplace to deploy FortiGate NVAs into your vWAN hub. Use "Fortinet FortiGate Security for Azure Virtual WAN" offer from Marketplace 
and remember to use data from your environment (region, resource group name, etc.) when deploying.

***Important:*** Do use your hub number (eg. "hub01") as FortiGate name prefix and as application name.
