## Deploy NVAs

Use Azure Marketplace to deploy FortiGate NVAs into your vWAN hub. Use "Fortinet FortiGate Security for Azure Virtual WAN" offer from Marketplace.

### Basics

Use data from your environment (region, resource group name, etc.) when deploying. You can use any FortiGate username and password, but remember that Azure restricts username and *"admin"* is not allowed. You can also copy the same credentials you use for other systems in this lab.
