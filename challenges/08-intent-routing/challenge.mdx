## Enabling Azure intent routing

In Azure web UI go to **Virtual WANs** → **vwanlab-vwan-common** → **Connectivity** → **Hubs** → ***your hub***

In **Routing** → **Routing Intent and Routing Policies** enable routing of **Private traffic** through your *Network Virtual Appliance*

> Note: do not change settings for **Internet traffic**. Leave it empty.
