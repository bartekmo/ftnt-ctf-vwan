## Enabling FGSP

Initially both FortiGates are configured as stand-alone. In some situations where traffic flows might be asymmetrical you need to add FGSP configuration to your cluster. The example CLI code:

```
config system standalone-cluster 
  set standalone-group-id 1 
  set group-member-id 1 
  config cluster-peer 
    edit 0 
    set peerip x.x.x.peer 
    next 
  end 
end 

config system ha 
  set session-pickup enable 
  set session-pickup-nat enable 
  set session-pickup-connectionless enable 
  set override disable 
end
```

You can implement the above settings manually on each individual FortiGate or you can use FortiManager. You'll get points once `diag sys ha standalone-peers` shows FGSP peer.
