Before the FortiGate NVAs can pass traffic, they must be licensed using **FortiFlex** tokens. Each NVA requires its own token.

## Your FortiFlex tokens

<EnvVar field="flex_token1" label="FortiFlex Token — NVA 1" secret />
<EnvVar field="flex_token2" label="FortiFlex Token — NVA 2" secret />

## NVA management access

Access each FortiGate NVA through its public IP address:

<EnvVar field="fgt_nva1_pip" label="NVA 1 Public IP" />
<EnvVar field="fgt_nva2_pip" label="NVA 2 Public IP" />

Log in with username **admin** and the password provided by the trainer.

## Steps

1. Open https://<EnvVarInline field="fgt_nva1_pip" /> in your browser
2. Navigate to **System → FortiGuard**
3. Under **License Information**, click **Enter License**
4. Paste **Token 1** from above and confirm
5. Wait for the license to activate — the status should change to **Licensed**
6. Repeat steps 1–5 for NVA 2 using **Token 2** and the address https://<EnvVarInline field="fgt_nva2_pip" />

## Verification

Once both NVAs are licensed, run the following CLI command on each to confirm:

```
get system status
```

Look for `License Status: Valid` in the output.

> **This challenge is scored.** The prober checks that both NVAs have a valid license before awarding points.
