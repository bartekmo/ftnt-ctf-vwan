Before the FortiGate NVAs can pass traffic, they must be licensed using. In this lab we are using **FortiFlex** licensing. Licenses are already generated and you will not be accessing FortiFlex portal. The license tokens for your two FortiGates are as below. If the do not work, please ask workshops trainer for assistance.

## NVA management access

You can access your FortiGates using SSH or Web UI for this task. You configured your credentials during deployment in the previous step.

<NvaCard />

## Licensing using Web UI

Connect to each FortiGate web ui using the links above. After a short while you will be asked for license. Select **FortiFlex token** and paste token from the list above (note: each token can be used only once).

![](lic-web.png)

## Licensing using SSH 

After connecting run the command `execute vm-license` followed by the FortiFlex token. The appliance will reboot to activate the license.

![](lic-ssh.png)


## Verification

Once both NVAs are licensed they will connect to FortiManager. Lab automation will authorize them and move to your ADOM. Please verify if your NVAs show up in your FortiManager ADOM.
![](fmg.png)
